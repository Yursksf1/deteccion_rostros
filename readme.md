
0) download project git
```
git clone <url_git>
```

1) create virtualenv 
```
pip install virtualenv
virtualenv -p python3 myenv
```
2) activate enviroment
```
# Unix (linux o mac)
source myenv/bin/activate

# Windows
source myenv/Scripts/activate.bat
```

3) install requirements
```
pip install requirements.txt
```

4) run code 
```
python example.py
```
